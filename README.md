# iPhone Service

You can see it [here](http://paketik7gm.temp.swtest.ru/)

**How to make it work?**

Install the dependencies:
```shell
npm install
```
Running a web server:
```shell
gulp
```

Deploy on hosting:
```shell
gulp deploy
```
